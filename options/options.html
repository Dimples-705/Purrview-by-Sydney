Options

By default the extension uses the free, offline extractive summarizer. To enable abstractive mode fully offline, bundle Transformers.js (see instructions below).

Enable Abstractive (Transformers.js)

  1. Download transformers.min.js from the official Transformers.js releases (Xenova) and place it at vendor/transformers.min.js in this extension directory.
  2. No API keys required. Models are downloaded on first use and cached by your browser.

Optional: Remote LLM (if you have an API key)

This extension is fully usable without any API, but if you prefer a hosted provider, you can add your key below and future versions could call it instead.

Privacy

Offline modes do not send page content anywhere. If you later enable a remote LLM, content will be sent to that provider per their terms.
